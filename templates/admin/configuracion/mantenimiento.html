📤 Subir backup desde tu computadora

Si tenés un backup guardado en tu PC, Dropbox, OneDrive o disco externo, podés subirlo acá para restaurarlo.

Solo se aceptan archivos .db

💡 ¿Cómo recuperar un backup externo?

Si guardaste backups en Dropbox u OneDrive:
1. Abrí Dropbox/OneDrive en tu computadora   2. Entrá a la carpeta "AMPARO Backups"   3. Descargá el archivo .db que querés restaurar   4. Subilo acá con el botón de arriba

Si recibiste el backup por WeTransfer:
1. Abrí el email con el link de WeTransfer   2. Descargá el archivo .db   3. Subilo acá con el botón de arriba

⚠️ Después de restaurar reiniciá la aplicación para que los cambios tomen efecto.
